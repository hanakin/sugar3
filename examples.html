Introducing Sugar3 A new forward thinking, extremely scalable, dynamically lightweight front-end framwork. Based on Atomic Design and SMACSS principals.

Download

  • Structurized Layer Approach

    Built from several layers each adding another peice to the puzzle. Completeing the modularity of the framework.

  • Atomic Design Stratagy

    Atoms are the basic building bodys of matter. Applied to web interfaces, the small resuable styles we use more often by themselves

  • Responive Mobile First

    Based on the Bootstrap implementation of a responsive grid and utilities. Remastered for better scalability and extensability.

  • Explicit Documentation

    Consistent informative Documentation for all clasess following idiomatic principles set forth by Necolas Gallagher

  • Dynamic, Light & Powerful

    Completely modular down to the individual set of atoms. You decide just what you need for each and every page/project.

  • Endless Scalability

    Unlike most frameworks we don't overload you with prebuilt modules. Instead we provide a base workflow to develop your own library.

The Core Layer

Consists of Three parts, laying the ground work for all ofyour projects.

Common

Set some project specific styling for all the HTML components per project.

Layout

A responsive forward think grid system used alongside powerful utilities to align and position elements for all devices.

Atoms

Small useful classes on their own, equally effective at extending functionality to modules or components.

The Sugar Layer

Consists of Two parts, build your library of objects in your web arsenal.

Modules

Series of dynamically accessible commonly used elements that you might use in various projects.

Components

Series of dynamically accessible commonly used blocks or sections of modules layed out in various ways.

Theme

The Theme Layer
